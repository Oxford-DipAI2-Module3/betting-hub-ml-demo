🚀 Implementation Requirements

Phase 1: Impression Tracking (URGENT)

Implement Event: discover_tile_impression

When to fire: Once per tile per session when >50% visible in viewport

Deduplication: Session-level (Firebase handles this automatically)

Required Parameters:

  • tile_id - Join to clicks
  • tile_position - Array index (1-30), NOT "order" field
  • tile_expiry_timestamp - From API "end" field ⚠️ NEW
  • tile_start_timestamp - From API "start" field ⚠️ NEW
  • bookmaker, bet_type, sport, tile_type - From API keywords
  • user_id, session_id, timestamp - From analytics

Timeline: 3-5 days implementation + 4 weeks data collection
